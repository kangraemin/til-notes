# TIL Notes

개인 TIL(Today I Learned) 기록 저장소.

[mcp-learning](https://github.com/kangraemin/mcp-learning) MCP 서버와 연동하여 사용한다.

## 사용법

Claude Code에서 자연어로 말하면 자동으로 저장된다.

```
오늘 Python 데코레이터 배웠어. 기록해줘.
방금 토큰 최적화 논의한 거 정리해줘.
이번 주 뭐 배웠는지 보여줘.
```

## 구조

```
tils/
  YYYY-MM-DD-{slug}.md   ← 각 TIL 파일
```
